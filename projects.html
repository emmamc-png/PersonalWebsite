About the Dungeon Game!

This game focused on utilizing different data structures to create a dungeon crawler game. The game mainly focused on Graphs to find the BFS and DFS of the dungeon.

Additionally, there was a focus on utilizing the Stack in C++ for pointers and dynamic memory allocation. There was also the use of a hash table which incorporated chaining through linked lists. Overall, this project taught me a lot about different data structres and how to manage them!

About my BlackJack Game!

This game was the cumulative project for my Computer Architecture and Assembly course. It was written in x86 Assembly language and focused on using low-level programming to create a BlackJack game that included multiple decks. The game started off by giving the player their two original cards and allowed them to either hit or stay. Then the dealer would be automatically played through the program which included a limit of the dealer not hitting beyond 17. Finally, the winner would be determined and the player could choose to play again or exit the game.

This project taught me a lot about managing memory and registers in low-level programming, especially when the user chose to play again and having to clear out the previous game data.
